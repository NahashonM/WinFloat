# ![WinFloat](pin.ico) WinFloat

WinFloat is a simple productivity tool that lets you pin any window so it is always stays on top of other windows.  

Whether you’re taking notes from a video or keeping a calculator handy, WinFloat ensures your most important apps never get buried under other windows, even when you're clicking between different tasks.

<br/>

## Features

- **Crosshair picker** — Use a crosshair to select any window to be pinned. Press **Escape** at any time during selection to cancel.
- **Pin or Unpin by hotkey** — Use configurable shortcuts to pin or unpin the active foreground window.

<br/>



## Usage

#### Pinning Windows using shortcut
1. Focus the window you want to pin on top.
1. press the **Pin** shortcut (default: **Ctrl+Alt+Shift+P**).

#### Pinning Windows using crosshair
1. Click and hold the WinFloat crosshair.
1. Drag the cursor over the window you want to pin and then release it while still over the window.  

_Press **Escape** to cancel selection._

#### UnPinning Windows
1. Focus the pinned window you want to unpin.
1. press the **Unpin** shortcut (default: **Ctrl+Alt+Shift+U**).

#### Tips
WinFloat has a pin icon in the title bar that can be used to toggle the pin state of the app.

<br/>

## Settings Description

| Setting | Description   |
|--------|--------------------|
| Pin window shortcut | Set the pin shorcut. |
| Unpin window shortcut | Set the unpin shorcut. |
| Minimize to tray on exit | When enabled, closing the window sends it to the system tray instead of quitting.        |
| Crosshair can target this window | When enabled, the crosshair picker can pin WinFloat’s own window. |
| Default | Reset the settings to default. |
| Save | Save changes to config file. |

Settings are stored under: `%LocalAppData%\WinFloat\config.cfg`.
